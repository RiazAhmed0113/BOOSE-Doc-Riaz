Namespace BOOSE_App.Tests

Classes

AppArrayTests

Unit tests for the AppArray preprocessor handler. Each test verifies that the handler correctly declares arrays of the requested type and size and updates the provided PreprocessorState accordingly.

AppBooleanTests

Unit tests for the AppBoolean preprocessor handler. Each test verifies that boolean declarations and initializations are handled correctly and that state mutation in PreprocessorState matches expectations.

AppCanvasTests

Unit tests for the AppCanvas class. Verifies basic drawing API behaviour: moving the pen, drawing lines/shapes and producing a bitmap.

AppCircleTests

Unit tests for the AppCircle command and basic parser/command integration. Ensures a small program containing a circle command executes without throwing and produces a bitmap.

AppForTests

Unit tests for the AppFor preprocessor block handler. Verifies the FOR...END FOR unrolling behaviour and integer-argument evaluation.

AppIfsTests

Unit tests for the AppIfs preprocessor block handler. Tests verify that IF/ELSE sections are selected and processed correctly given prepopulated state.

AppIntTests

Unit tests for the AppInt preprocessor handler. Verifies integer declaration and initialization behaviour and the resulting mutations to PreprocessorState.

AppMethodTests

Unit tests for the AppMethod preprocessor handler. Verifies method declaration parsing, registration and that a subsequent "call" invocation executes the method body at preprocess time and stores the return value in state.

AppPeekTests

Unit tests for the AppPeek preprocessor handler. Verifies that values can be read from integer and real arrays and stored into declared scalar variables in PreprocessorState.

AppPokeTests

Unit tests for the AppPoke preprocessor handler. Verifies behaviour for writing values into integer and real arrays in PreprocessorState.

AppRealTests

Unit tests for the AppReal preprocessor handler. Verifies real variable declaration and initialization behaviour and the resulting mutations to PreprocessorState.

AppRect_MoveTo_Tests

Unit tests for the BOOSE_App.AppRect and AppMoveTo commands via parser integration. Each test runs a small BOOSE program against a fresh AppCanvas and asserts expected canvas state or output.

AppWhileTests

Unit tests for the AppWhile preprocessor block handler. Ensures WHILE blocks are evaluated and their bodies emitted repeatedly until the condition becomes false.

ExpressionEvaluatorTests

Unit tests for the ExpressionEvaluator class. Verifies numeric expression evaluation and boolean expression evaluation behavior using a PreprocessorState instance as the variable source.

PreprocessorTests

Unit tests for the preprocessing stage of the BOOSE interpreter. Exercises scalar declarations, loop/unrolling behaviour, method registration/calls and WHILE block processing.
