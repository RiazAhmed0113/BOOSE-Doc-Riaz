Handles preprocessing lines that declare or initialize real (double) variables:

  • "real name" declares a real variable with initial value 0.0
  • "real name = <expr>" declares (or updates) a real variable with the evaluated expression value

TryHandle(string, IPreprocessorState, IExpressionEvaluator, out string?, out string?)

Attempt to handle a raw source line during preprocessing.

public bool TryHandle(string line, IPreprocessorState state, IExpressionEvaluator evaluator, out string? outProcessedLine, out string? outError)

Parameters

line string

Raw source line to inspect.

state IPreprocessorState

Current IPreprocessorState. Handlers may read from and mutate this state (declare variables, update arrays, set temporaries, etc.).

evaluator IExpressionEvaluator

IExpressionEvaluator instance to evaluate numeric or boolean expressions when needed.

outProcessedLine string

When the method returns true this out parameter indicates the line to forward to the parser. A null value means the handler consumed the line and it should be dropped.

outError string

When the method returns true and an error occurred during handling, this contains a user-facing error message. Otherwise it should be null.

Returns

bool

true if the handler recognized the line (even if an error occurred), otherwise false. Returning false indicates the preprocessor should try other handlers.
