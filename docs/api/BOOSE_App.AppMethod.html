Remarks

  • TryHandleBlock(string[], ref int, IPreprocessorState, IExpressionEvaluator, out List<string>, out string?, int) parses and registers METHOD declarations in the provided IPreprocessorState (adds entries to Methods).
  • TryHandle(string, IPreprocessorState, IExpressionEvaluator, out string?, out string?) handles runtime-style "call" lines by binding parameters, executing the method body at preprocess time and storing the return value.

TryHandle(string, IPreprocessorState, IExpressionEvaluator, out string?, out string?)

Try to handle a single "call" invocation line. When a call is handled the method body is executed at preprocess time with parameters bound into temporary variables, the method return is stored in the preprocessor state and the call line is consumed (not forwarded).

public bool TryHandle(string line, IPreprocessorState state, IExpressionEvaluator evaluator, out string? outProcessedLine, out string? outError)

Parameters

line string

The raw source line to inspect (expected form: "call name [arg1 arg2 ...]").

state IPreprocessorState

Current preprocessor state containing variable tables and registered methods.

evaluator IExpressionEvaluator

Expression evaluator used to evaluate parameter and assignment expressions.

outProcessedLine string

When handled: processed output line to forward to the parser; a null value indicates the call was consumed by the preprocessor. When not handled: remains null and the method returns false.

outError string

On failure contains an error message describing the problem; otherwise null.

Returns

bool

true if the line was recognized as a call (even if it produced an error), otherwise false.

TryHandleBlock(string[], ref int, IPreprocessorState, IExpressionEvaluator, out List<string>, out string?, int)

Parse a METHOD ... END METHOD block beginning at index in lines. On success the method declaration is registered in state.Methods and the index is advanced to the END METHOD line.

public static bool TryHandleBlock(string[] lines, ref int index, IPreprocessorState state, IExpressionEvaluator eval, out List<string> outProcessed, out string? outError, int baseLineNo = 1)

Parameters

lines string[]

The full source lines array (normalized by the caller).

index int

Index of the starting line; on return this is advanced to the matching END METHOD index when a METHOD block is handled.

state IPreprocessorState

Preprocessor state used to register the method return variable and to store the method declaration.

eval IExpressionEvaluator

Expression evaluator (header parsing does not use the evaluator but it is provided for API consistency).

outProcessed List<string>

Output list of processed lines produced by the block. For METHOD blocks this list is normally empty.

outError string

When parsing fails contains a concise, user-facing error message; otherwise null.

baseLineNo int

Base line number offset used when producing user-facing error messages (default 1).

Returns

bool

true if a METHOD block was recognized at index (even if an error occurred), otherwise false. When true the caller should not attempt other handlers for the same lines.
