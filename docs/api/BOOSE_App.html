Namespace BOOSE_App

Classes

AppArray

Handles array declarations of the form: array int name = <sizeExpr> array real name = <sizeExpr> Also accepts the shorthand without '=': array int name <sizeExpr> Creates and registers the array in the preprocessor state.

AppBoolean

Preprocessor line handler for boolean declarations and optional initialization. Recognises lines of the form:

  • "boolean <name>" (declare)
  • "boolean <name> = <expr>" (declare and initialise) When handled the declaration is consumed by the preprocessor (no runtime output) and the named boolean variable is created/updated in the provided IPreprocessorState.
AppCanvas

Simple bitmap canvas implementation of ICanvas from BOOSE Provides basic drawing methods like moveto, drawto, circle, rect, tri, writetext.

AppCanvasSingleton

Simple singleton holder for AppCanvas. Provides a non-invasive wrapper so the application can obtain a single shared canvas instance without changing the concrete AppCanvas implementation.

AppCircle

Circle command to enable unrestricted circle drawing. Lets user input higher radius values.

AppCommandFactory

Factory that creates application-specific BOOSE.ICommand instances. Returns application-provided command implementations for commands the application overrides (for example "circle", "rect", "moveto") and delegates to the base BOOSE.CommandFactory for any other command names.

AppFor

Preprocessor block handler for FOR...END FOR constructs. The handler unrolls FOR loops at preprocess time, updating IPreprocessorState and emitting the runtime lines that should be executed for each iteration.

AppIfs

Simple block handler for IF / ELSEIF / ELSE / END IF constructs used by the preprocessor.

AppInt

Preprocessor line handler for integer declarations and optional initialization. Recognises lines of the form:

  • "int <name>" (declare)
  • "int <name> = <expr>" (declare and initialise) When handled the declaration is consumed by the preprocessor (no runtime output) and the named integer variable is created/updated in the provided IPreprocessorState.
AppMethod

Preprocessor handler for METHOD ... END METHOD blocks and CALL invocations.

AppMoveTo

Command that moves the current pen position to a specifc coordinate on the canvas

AppPeek

Handles "peek target = arrName index" preprocessing lines. Reads an element from an integer or real array and stores it in a declared scalar variable.

AppPoke

Handles preprocessing lines of the form poke <array> <indexExpr> = <expr>. Evaluates the index and expression, and writes the computed value into the named integer or real array in the provided IPreprocessorState. This handler consumes the source line (does not forward it).

AppReal

Handles preprocessing lines that declare or initialize real (double) variables:

  • "real name" declares a real variable with initial value 0.0
  • "real name = <expr>" declares (or updates) a real variable with the evaluated expression value
AppWhile

Simple block handler for WHILE ... END WHILE Evaluates the boolean condition and processes the loop body repeatedly at preprocess time, updating IPreprocessorState and emitting the runtime lines produced each iteration.

ExpressionEvaluator

Small expression evaluator used by preprocessing handlers. Implements IExpressionEvaluator.

Form1

Main application form for the BOOSE interpreter UI. Hosts the program inout textbox, a run button, an error log and a picture box that displays the drawing canvas.

PreprocessorState

Shared state used by preprocessing handlers. Implements IPreprocessorState.

VariableProcessor

Preprocessing helper that walks raw program text, handles declarations/blocks and emits a preprocessed program suitable for the parser.

Interfaces

IExpressionEvaluator

Abstraction for expression evaluation used by handlers and the preprocessor.

ILineHandler

Represents a preprocessing line handler. Implementations inspect a single source line and may consume it (perform actions against the IPreprocessorState) or produce a transformed line to be forwarded to the parser.

IPreprocessorState

Abstraction for the state used by preprocessing handlers.
