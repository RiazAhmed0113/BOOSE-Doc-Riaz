Property Value

Dictionary<string, (string ReturnType, string Name, List<(string Type, string Name)> Parameters, List<string> BodyLines)>

Remarks

The value tuple layout is: (ReturnType, Name, Parameters, BodyLines).

  • ReturnType: string ("int", "real", "boolean")
  • Name: method name
  • Parameters: list of (Type, Name) tuples for each parameter
  • BodyLines: original source lines that form the method body The dictionary is case-insensitive on keys. It may be null when no methods are registered.
