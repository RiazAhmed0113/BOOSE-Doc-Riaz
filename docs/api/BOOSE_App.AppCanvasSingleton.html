Simple singleton holder for AppCanvas. Provides a non-invasive wrapper so the application can obtain a single shared canvas instance without changing the concrete AppCanvas implementation.

Remarks

Call Initialize(int, int) once at application startup (for example in the main form constructor) to create the internal AppCanvas instance. After initialization, access the shared instance via the Instance property.
